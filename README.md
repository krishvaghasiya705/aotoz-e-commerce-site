## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- MongoDB

### Installation

1. Clone the repository
```sh
git clone https://github.com/krishvaghasiya705/aotoz.git
```

2. Install dependencies
```sh
cd aotoz
npm install
```

3. Set up environment variables
```sh
cp .env.example .env
```

4. Start the development server
```sh
npm run dev
```
